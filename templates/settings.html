Windows Startup

Configure the Project Manager to start automatically when Windows boots.

Setup Instructions:
  1. Run the PowerShell script install_startup_task.ps1 as Administrator
  2. Or manually add a shortcut to run_manager.bat to your Startup folder
  3. The manager will start on Windows login
Startup Script

Creates a scheduled task for auto-start.

install_startup_task.ps1
Manual Start

Run to start the manager manually.

run_manager.bat
API Information

REST API for external control (e.g., OpenClaw integration).

Endpoint Method Description
/api/projects GET/POST List/Create projects
/api/projects/<id>/start POST Start project
/api/projects/<id>/stop POST Stop project
/api/projects/<id>/toggle-mode POST Toggle auto/manual
/api/system/summary GET System summary
Base URL: http://localhost:8787
